# prueba fullstack
fullstack developer

# Instalación

```
npm install
```
# Compilación

```
npm run build
```
# Ejecución

```
npm run dev
```
### una vez teniendo instaladas las dependencias podemos realizar las peticiones de get, post, delete y update

# Dependencias
 * @babel/polyfill
 * express
 * morgan
 * pg pg-hstore
 * sequelize

# Dependencias dev
 * @babel/cli @babel/core @babel/node @babel/preset-env
 * nodemon
